#Gestión de versiones y colaboración con Git y GitHub

#Descripción del proyecto:

El presente proyecto tiene como propósito documentar el proceso de gestión de versiones y colaboración utilizando Git y GitHub. A lo largo del desarrollo, se emplearon diversos comandos fundamentales que permitieron crear, configurar y administrar un repositorio de manera estructurada.

El objetivo principal fue comprender y aplicar los pasos esenciales para el control de versiones, desde la creación de carpetas y la inicialización del repositorio, hasta la conexión con un repositorio remoto y la realización de confirmaciones de cambios (commits). Además, se busca fortalecer las habilidades prácticas para el trabajo colaborativo en entornos de desarrollo, aplicando buenas prácticas de seguimiento y actualización del código.

Este proyecto refleja la aplicación práctica de los conocimientos adquiridos durante el curso, fomentando una comprensión más sólida sobre la importancia del control de versiones y la colaboración.
